# My Fridge Planner

- 2021 마포구청 뉴딜일자리 사업단
- 서비스 기획 및 개발 연습 토이 프로젝트

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
